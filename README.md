# Terror Idle: Main God Space

Jogo idle clicker inspirado em **Infinite Horror (无限恐怖)**, com mecânicas de Cookie Clicker, Clicker Heroes, Tap Titans 2 e Antimatter Dimensions.

## Como jogar

- **Clique** para atacar monstros e ganhar Reward Points (RP)
- **Contrate heróis** no Salão do Deus Principal para gerar DPS passivo
- **Avance zonas** matando monstros — boss a cada 10 zonas, Mega Boss a cada 100
- **Modo ⚔ AVANÇANDO** → tenta matar o boss no timer. Se falhar, cai pro modo farm
- **Modo 🌾 FARMANDO** → acumula RP na última zona segura até ficar forte o suficiente
- **Faça Prestige (Ciclo)** para ganhar Hero Souls e começar mais forte
- **Transcenda** após 10 ciclos para ganhar Void Essence e multiplicadores permanentes
- **Bloodlines, Dimensões, Ancients** desbloqueiam progressivamente

## Moedas

| Moeda | Como ganhar | Para que serve |
|---|---|---|
| Reward Points (RP) | Matar monstros | Comprar heróis e upgrades |
| Hero Souls | Prestige | Comprar Ancients |
| Gene Frags | Mega Boss / Prestige | Dimensões e Shop |
| Rank Points | Passivo / Boss | Shop permanente |
| Void Essence | Transcendência | Bloodlines |

## Rodar localmente

```bash
pnpm install
pnpm dev
```

Abra [http://localhost:3000](http://localhost:3000).
